# Retail Sales Intelligence Dashboard

## Project Overview

This project analyzes 25,000+ global retail orders and $12.64M in sales data using SQL and Power BI.

The objective was to identify revenue drivers, profitability patterns, category performance, and product-level optimization opportunities.

## Tools Used

* SQL (MySQL)
* Power BI
* Excel

## Key Findings

* Central region generated the highest revenue ($2.82M)
* Technology category delivered the highest revenue ($4.74M)
* Canada achieved the highest profit margin (26.62%)
* Multiple products generated significant losses
* Overall business profit margin was 11.61%

## Dashboard Pages

### Page 1: Executive Overview

KPIs, revenue trends, regional performance, category contribution.

### Page 2: Regional Performance Analysis

Regional revenue, profit, profit margin, and geographic analysis.

### Page 3: Product & Category Analysis

Category performance, top products, and loss-making products.

### Page 4: Executive Insights

Strategic recommendations and business actions.

## Business Impact

The analysis identified opportunities to improve profitability through regional optimization, category management, and product portfolio decisions.
